## Installation

```bash
# Clone the repository
git clone https://github.com/CodinWaffle/Stride.git

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env

# Initialize the database
flask db upgrade

# Run the application
flask run
```
